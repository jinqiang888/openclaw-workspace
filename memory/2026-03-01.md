### 飞书消息输出 null 问题 🎯 已定位

**症状**: AppFlow 显示成功，飞书收不到消息，输出 null

**根本原因**: 
- 用户提供的是**阿里云 AppFlow Webhook**，不是飞书官方 webhook
- AppFlow 流程内部配置错误，没有正确调用飞书 API 或没有返回值
- 测试返回：`{"success":false,"code":"FlowRunFailed","message":"Flow Run Failed"}`

**解决方案**:
1. **推荐**: 获取飞书官方 Webhook，直接使用 `feishu-message-guard` 技能
2. **备选**: 修复 AppFlow 流程配置，添加返回动作

**已创建文档**: `appflow_diagnosis.md` - 完整分析和解决方案
